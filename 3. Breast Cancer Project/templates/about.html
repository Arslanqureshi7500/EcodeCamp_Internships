About This Project

This project analyzes the Breast Cancer Wisconsin (Diagnostic) dataset to classify breast tumors as either malignant or benign based on various features extracted from digital images.

We have implemented several models, including Logistic Regression, Decision Tree, Random Forest, and Gradient Boosting Classifier, to evaluate their performance on the dataset.

Breast Cancer Wisconsin (Diagnostic) Dataset Overview

The Breast Cancer Wisconsin (Diagnostic) dataset is used to predict whether breast cancer tumors are malignant (cancerous) or benign (non-cancerous). The dataset contains the following features:

  • Radius: Mean of distances from the center to points on the perimeter.
  • Texture: Standard deviation of gray-scale values.
  • Perimeter: Length around the tumor's boundary.
  • Area: Area size of the tumor.
  • Smoothness: Local variation in radius lengths.
  • Compactness: Perimeter² / area - 1.0.
  • Concavity: Severity of concave portions of the contour.
  • Concave points: Number of concave portions of the contour.
  • Symmetry: Symmetry of the tumor's shape.
  • Fractal dimension: "Coastline approximation"—how detail changes with scale.
